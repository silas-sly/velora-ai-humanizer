# Velora AI Humanizer

Free AI humanizer website for rewriting robotic text into natural, clear writing. Runs entirely in the browser with no signup and no server.

## Quick start (test locally)

Open `index.html` in your browser.

## Deploy to GitHub Pages (free hosting)

### 1. Create a GitHub account

If you don't have one, sign up at [github.com](https://github.com) (free).

### 2. Create a new repository

1. Click **New repository** on GitHub
2. Name it `velora-ai-humanizer` (or any name you like)
3. Leave it **Public**
4. Do **not** add a README (you already have one)
5. Click **Create repository**

### 3. Push this project

Replace `YOUR_USERNAME` with your GitHub username and run these commands in the project folder:

```bash
cd ~/Projects/study-snap
git branch -M main
git add .
git commit -m "Launch Velora AI Humanizer site"
git remote add origin https://github.com/YOUR_USERNAME/velora-ai-humanizer.git
git push -u origin main
```

### 4. Enable GitHub Pages

1. On GitHub, open your repo → **Settings** → **Pages**
2. Under **Build and deployment**, set Source to **Deploy from a branch**
3. Branch: **main**, folder: **/ (root)**
4. Click **Save**
5. Wait 1–2 minutes — your site will be live at:

   `https://YOUR_USERNAME.github.io/velora-ai-humanizer/`

## Customize before sharing

1. **Email:** In `index.html`, replace `your-email@example.com` with your real email.
2. **Analytics:** Uncomment the Google Analytics block and replace `G-XXXXXXXXXX`.
3. **Domain metadata:** Replace `https://example.com/` in `index.html`, `robots.txt`, and `sitemap.xml` with your real site URL.
4. **Sponsor slots:** Keep two sponsor cards priced at $10 for 14 days each.
